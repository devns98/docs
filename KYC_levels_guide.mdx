# Forte KYC Widget – Product Guide

This guide helps developers quickly visualize how the Forte KYC widget behaves at different KYC Levels. Each level unlocks a new set of identity requirements and UI steps.

## KYC Level 1 – Email Verification

### <Icon icon="badge-check" size={22} /> Identity Requirements:

- Valid email address
- Wallet screening

### <Icon icon="image" size={22} /> UI Screens:

1. Email input
2. OTP verification

![Email Input](https://your-image-path.com/kyc1-email.png)

\
![OTP Verification](https://your-image-path.com/kyc1-otp.png)

---

## 🔁 KYC Level 2 – KYC Lite

### 🔐 Identity Requirements:

- First & last name
- Verified phone number
- Self-attested country

### 🖼️ UI Screens:

1. Full name
2. Phone input & verification (OTP)

![Identity Preview](https://your-image-path.com/kyc2-preview.png)

\
![Phone Input](https://your-image-path.com/kyc2-phone.png)

\
![Success](https://your-image-path.com/kyc2-success.png)

---

## 🔒 KYC Level 3 – Standard KYC

### 🔐 Identity Requirements:

- Email, Full Name, Verified Phone Number (Level 2)
- Residential address
- Government-issued ID
- Live selfie

### 🖼️ UI Screens:

1. Identity Preview for missing vectors
2. Address input
3. QR code for verification flow link (mobile)
4. Success / Pending / Rejected

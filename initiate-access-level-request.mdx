Send a `POST` request to `/compliance/v2/kyc` to begin the verification flow.

### Headers

```
Authorization: Bearer {access_token}
```

### Request

Pass the desired `level` (1-3) for the user's wallet to complete verification for via the Forte Compliance widget. For more information about `level` requirements see [Verification Flows](./access_levels_flows).

```json
{
  "action": {
    "type": "OCC_RULES_ENGINE_V2",
    "level": 3
  },
  "customer": {
    "wallet": {
      "blockchain": "base_sepolia",
      "address": "0xF84D68F29664F907eE9d03E534105b7aA6045A"
    },
    "external_id": "0xF84D68F29664F907eE9d03E534105b7aA6045A"
  }
}
```

### Response

```json
{
  "data": {
    "flow": "KYC_OCC",
    "error_code": null,
    "widget_data": "<widget-data>"
  }
}
```

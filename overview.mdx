Welcome to the Forte KYC and Compliance SDK integration guide.

This guide outlines how to integrate Forte's KYC functionality using a simple widget-based approach. The integration is designed to work seamlessly with Forte's Rules Engine.

**Integration consists of three parts:**

1. [Obtain Access Token](./obtain-access-token)
2. [Initiate KYC Request](./initiate-kyc-request)
3. [Integrate KYC Widget](./integrate-widget)

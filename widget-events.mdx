The Forte Compliance widget will emit client-side events that provide information on when a user closes the widget. These events help you to handle post-verification flow UX as desired.

## Event Types

Not attached to docs.json.

## Event Listener Snippet

```javascript
window.addEventListener('ForteKYCSuccess', e =>
        // e.detail has the paymentIntentId

        )
      );
```
